Manual Setup Package for Modnix

Modnix is a program for loading mods into Phoenix Point.

Generally, you should use the Installer first. But when things don't work out, follow these steps for a manual setup:

  1. Reboot computer. Always reboot first.
  2. Find "game code folder". Default to 
    C:\Program Files\Epic Games\PhoenixPoint\PhoenixPointWin64_Data\Managed
    . You can find 
    Assembly-CSharp.dll
    in it.
  3. Delete all files in the dll folder. Then go up two folders to find 
    UnityPlayer.dll
    , again deleting all dlls.
  4. From the game launcher (EGS or Steam), do Verify Files.
  5. Launch the game and make sure it works. Exit game.
  6. Explore 
    %UserProfile%\Documents
    , and creates the folders 
    My Games\Phoenix Point\Mods
    . This is the "mod folder".
  7. Move extracted 
    Modnix.exe
    into "mod folder".
  8. Move all files in the extracted 
    Managed
    folder into the "game code folder". Nothing should be overwritten if you cleared the folders in step 3.
  9. Double-click 
    ModnixInjector.exe
    , which has just been moved. It should says "Injection complete!"
  10. Basic setup complete. Launch the moved 
    Modnix.exe
    and see whether it works.

Some mods may be unsatisfied by a basic setup. See wiki for how to solve them, and/or other troubleshoot guides.
